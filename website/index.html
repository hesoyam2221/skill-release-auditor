Beta MVP

Catch skill launch bugs before your users do.

Skill Release Auditor checks an OpenClaw skill for README/code mismatches, broken installers, missing tests, release contamination, and beta blockers. Then it generates a clean report, launch checklist, and teammate handoff.

Get it on GitHub Get launch updates

What it does

  • Flags README vs code mismatches
  • Finds broken installer file references
  • Warns on broad file/path access heuristics
  • Detects missing tests and release contamination
  • Outputs status, checklist, handoff, and findings JSON

What it doesn't do yet

  • No semantic verification
  • No automatic upstream code fixes
  • No direct ClawHub publishing
  • No generated README rewrites in the free version
